Demo Proposal of MechArchon Game

Class Instructor Date Language Ta'ed Code
CS 4455 Game Design Blair Macintyre Fall 2012 Unity/C# No Code N/A

This was a demo of the Semester project for the senior level Game Design class. We were individually required to implement a Unity-based game demo that captured the flavor of the game we wished to implement, with the idea that this would be used to recruit other students to work on the team.

We only had a bit more than a week to implement this, which, considering the complexities of developing anything in Unity, was really a short period of time, but considering this I was pretty happy with my demo, and got two other good teamates to work with me on the final project implementation.
